Wavetable oscillator - an ultra-clean wavetable oscillator.

Features:

· Support for Serum wavetable format (WT frames each with 2048 samples length)
· Up to 256 waves in a single oscillator, morphing, 7 voices stereo unison
· Modulatable Position, audio-rate modulatable Phase and Tune
· Modular wavetable manipulation.

The Wavetable oscillator contains an internal rack for advanced wavetable effects, using additional modules like Wave effect and Wave filter. Tap on the rack icon in the bottom-right corner to open the internal rack that is used for additional wavetable processing.

Use the Wave effect and Wave filter modules from PROCESSOR / Spectral section.

41 great wavetables created by Echo Sound Works are included. You can also import standard wave files that are not dedicated wavetables (and therefore won't scan as smoothly).

Wavetable position: Sweeps through the wave shapes in the table.
Wavetable selector: Select the wavetable to use. The arrows move through the available wavetables. Tapping the name opens Drambo's file browser to select wavetables and to import your own wave tables. The waveform display shows the currently selected wave shape in the table.
Phase: Adjusts the starting point in the wave for the oscillator.
X-Fade: Toggles smooth interpolation between wave forms.
Edit: Opens the dialog to change Unison properties
    · Detune: The amount of frequency detune between the voices
    · Blend: The amount blend between the unison voices and the original voice
    · Stereo: Toggles between Stereo (highlighted) and Mono.
Tune: Coarse (semitone) tuning of the oscillator
Fine: Fine (cents) tuning of the oscillator
Unison: the number of detuned copies of the wave to output for fuller sound. See Edit above.
Note Source: Tap to connect to a suitable input such as Midi to CV module.
Trigger Source: Tap to connect to a suitable input such as Midi to CV module.
Wavetable Effects: Opens the Wavetable Effects. See below.
